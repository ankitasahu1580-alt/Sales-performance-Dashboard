# Sales Performance Dashboard - Analysis

## Project Overview
This Power BI dashboard provides a comprehensive analysis of sales performance for a retail business. It tracks key metrics like revenue, profit, and units sold to help stakeholders make data-driven decisions.

## Live Preview
![Dashboard Screenshot](Screenshot%202026-04-14%20171742.png)

## Key Features
- **Dynamic Slicers:** Filter data by Date, Region, Category, and Gender.
- **KPI Tracking:** High-level view of Total Revenue ($1.7M), Profit ($832K), and Customer Satisfaction (3.2).
- **Sales Trends:** Monthly trend analysis using Line Charts.
- **Product Performance:** Insights into top-performing product categories.

## Tools Used
- **Power BI Desktop:** For report authoring and data visualization.
- **Power Query:** For data transformation and cleaning.
- **DAX:** For calculating custom measures and KPIs.

## How to use
Download the `.pbix` file from this repository and open it in Power BI Desktop to explore the interactive features.
